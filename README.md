KodeKart is a simple console-based E-commerce system built using Java and MySQL.  
Users can register, log in, browse products, add items to cart, and place orders.  
Admins can manage products and view all orders.

## Features
- User Registration & Login
- View Products
- Search Products
- Add to Cart / Remove from Cart
- Place Order
- View Order History
- Admin: Add / Update / Delete Products
- Admin: View All Orders

## Technologies Used
- Java
- JDBC
- MySQL
- Maven
- Eclipse IDE

## How to Run
1. Import the project as a Maven project in Eclipse.
2. Create the database in MySQL and run the SQL table script.
3. Update database credentials in `DBConnection.java`.
4. Run the `Main.java` file to start the application.

## Description
This project demonstrates basic Java concepts such as OOP, Collections, Exception Handling, and JDBC CRUD operations for database interaction. It is a simple implementation of an E-commerce system in console mode.
